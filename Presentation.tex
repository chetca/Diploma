\begin{frame}\frametitle{Метод бросания лучей}
Метод бросания лучей (англ. raycasting, рейкастинг) - один из методов рендеринга в компьютерной графике,
при котором сцена строится на основе замеров пересечения лучей с визуализируемой поверхностью.
\begin{figure}[h!]
\begin{center}
\includegraphics[width=11cm]{rayc}\\
\end{center}
\end{figure}
\end{frame}
\begin{frame}\frametitle{История}
Метод бросания лучей в публикациях по компьютерной графике впервые был
применён в 1982 г. для отрисовки моделей конструктивной блочной геометрии\\

\begin{figure}[h!]
\begin{center}
\includegraphics[width=10cm]{ScottDRoth}\\
\end{center}
\end{figure}
Roth, Scott D. (February 1982), "Ray Casting for Modeling Solids", Computer Graphics and Image Processing Т. 18: 109–144
\end{frame}
\begin{frame}\frametitle{История}
Для ПК рейкастинг в 1992 г. популяризировала игра Wolfenstein 3D
\begin{figure}[h!]
\begin{center}
\includegraphics[width=10cm]{Wolf3D}\\
\end{center}
\end{figure}
\end{frame}
\begin{frame}\frametitle{История}
Wolfenstein 3D реализует игровую сцену двумерной моделью, которая визуализируется в 3D
\begin{figure}[h!]
\begin{center}
\includegraphics[width=10cm]{WolfLevel}\\
\end{center}
\end{figure}
\end{frame}
\begin{frame}\frametitle{Цели и задачи}
\begin{block}{\textbf{Цель работы}}
Создание кроссплатформеного псевдотрёхмерного движка (англ. engine) для
3D визуализации помещений и маршрутов в них по 2D плану.
\end{block}
\begin{block}{\textbf{Задачи исследования}}
\begin{itemize}
\item исследование и непосредственная реализация эффективного алгоритма отрисовки проекции трёхмерной сцены
\item модификация алгоритма рейкастинга для вещественных координат
\item обеспечение интерактивности и поиск маршрутов
\item платформа для возможной реализации дополнительных сервисов и кроссплатформености
\end{itemize}
\end{block}
\end{frame}

\begin{frame}\frametitle{Описание метода: основная идея}
\begin{minipage}{0.49 \linewidth}
\begin{figure}[h!]
\begin{center}
\includegraphics[width=5cm]{mainIdea}\\ %основная идея
\end{center}
\end{figure}
\end{minipage}
\hfill
\begin{minipage}{0.49 \linewidth}

$\overrightarrow{\Delta a} = \frac{\left|\overrightarrow{AB}\right|}{w}$\\
$\overrightarrow{a_{i+1}} = \overrightarrow{a_i} + \overrightarrow{\Delta a}$\\
$\overrightarrow{a_1} = \overrightarrow{A}$\\
$\overrightarrow{a_n}=\overrightarrow{B}$\\
$\overrightarrow{r_i} = \overrightarrow{a_i} - \overrightarrow{P}$\\

\begin{center}\textbf{Алгоритм рейкастинга}\end{center}
\textit{
$\text{для каждой } i \in [1,n]$:\\
\hphantom{мммм}$l \leftarrow \text{расстояние-до-стены} (\overrightarrow{P}, \overrightarrow{r_i})$\\
\hphantom{мммм}$h \leftarrow \text{высота-отрезка}(l)$\\
\hphantom{мммм}$\text{отобразить-отрезок}(i, h)$\\
}
\end{minipage}
\end{frame}
